# react-flavor-example

## Fetch resource

```
$ git clone https://github.com/yukitaka13.1110/react-flavor-example.git
```

## Install dependency

```
$ cd react-flavor-example
$ npm install
```

# Build

## Development

```
$ npm run dev
```

## Staging 

```
$ npm run stg
```

## Production

```
$ npm run prod
```
